## ğŸš€ Getting Started

### Prerequisites

Make sure you have the following installed:

- [Flutter SDK](https://flutter.dev/get-started/) (v3.0 or above)
- [Dart SDK](https://dart.dev/get-started/) (comes with Flutter)
- [Android Studio](https://developer.android.com/studio) or [VS Code](https://code.visualstudio.com/) with Flutter and Dart plugins
- An Android emulator or a physical device connected

### Running the App

1. Clone the repository:
   ```bash
   git clone https://github.com/AnsahFredd/dart_flutter_projects.git
   ```

2. Navigate into the project folder:
   ```bash
   cd dart_flutter_projects/todoapp
   ```

3. Install dependencies:
   ```bash
   flutter pub get
   ```

4. Run the app:
   ```bash
   flutter run
   ```
